# AVtar

AVtar is a team-built web application developed for a hackathon/project competition, where our team secured 3rd prize.

## Overview

The project was created as a collaborative software solution with a focus on building a functional web application using a traditional full-stack structure. It includes server-side routing, views, models, documentation, and frontend assets.

## My Contribution

As part of the team, I contributed to the development and presentation of the project, including frontend/backend integration support, debugging, feature implementation, and preparing the project for demonstration.

## Achievement

- Secured 3rd prize in the competition
- Built and presented as a team project
- Demonstrated practical full-stack development and collaboration

## Tech Stack

- JavaScript
- HTML
- CSS
- EJS
- Node.js
- Express.js

## Project Structure

- `models/` - Data models
- `routes/` - Application routes
- `views/` - EJS frontend views
- `docs/` - Project documentation
- `server.js` - Main server entry point
